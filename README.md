# NachoMusic

A simple music downloading and streaming application, NachoMusic aims to be an alternative to desktop Spotify.

## Why?

![image](https://community.spotify.com/t5/image/serverpage/image-id/127647iE603CAA8BE04916A/image-size/large?v=v2&px=999)

Desktop Spotify (on Windows) sucks, it has frequent errors (see above) and setting up local files is a buggy, unintuitive mess.
